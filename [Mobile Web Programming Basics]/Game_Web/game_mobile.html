MOBILE GAME

모바일 게임(Mobile game)이란 핸드폰을 비롯한 모바일 기기를 통해서 플레이하는 게임이다. 다른 플랫폼과는 달리 인원과 비용이 적게 들기 때문에 소형 개발사가 많이 건드려 보는 게임류 중 하나이다.

Clash of Clans

개발사 : SUPERCELL

장르 : SNG

심의등급 : X

발매일(iOS) : 2012년 7월 2일

플레이 방식 : 자신의 자원과 병력을 이용하여 상대방의 모든 건물을 파괴하는 공격 방식과 , 다른 유저들의 공격을 방어해내는 디펜스 방식이 있다.

평가 : 2014년 , 춘추전국시대를 겪고 있던 한국 모바일게임계를 강타한 전세계 1.5억명 이상의 사용자들의 게임이다. 과금 유도가 적고 과금으로 인해 밸런스 붕괴를 일으키는 요소가 없는 탓인지 과금유도가 심한 한국 게임보다 인기가 더 많았었다.






Angry Birds Series

개발사 : Rovio Entertainment

유통사 : X

장르 : FPS

심의등급 : X

발매일(iOS , Andriod) : 2011년

플레이 방식 : 여러 종류의 기능과 특징을 가지고 있는 새를 새총에 장전(?)시켜 사물을 파괴하여 다음스테이지로 넘어가는 방식.

평가 : iOS와 안드로이드 운영체제에서 전무후무한 말도 안되는 기록을 갈아치웠던 게임. 79개국 앱스토어에서 1위를 기록했으며 여러 굿즈 상품들이 나왔었다. 하지만 급격히 변하는 모바일게임계의 트렌드를 따라가지 못해 순위권에서는 떨어진 상태이다.



ABYSSRIUM

개발사 : IDLE idea Factory , MAF Games

유통사 : Cheetah Mobile

장르 : Idle게임 , 시뮬레이션

심의등급 : X

발매일(iOs , Andriod) : 2016년 7월 15일

플레이 방식 : 게임을 시작하면 '외로운 산호석' 하나만 홀로 나타난다. 튜토리얼에 따라 화면을 탭 하면 산호석이 게임 내에서의 화폐인 생명력을 생산하며, 모은 생명력으로 물고기를 창조하여 산호석을 외롭지 않게 만들어 주면 되는 게임이다.

평가 : 특별한 조작없이 자동적으로 자원이 생산되는 방치형 게임이다. 특색있는 그래픽과 서정적인 음악은 게임을 하면서 힐링을 한다는 느낌을 받는다는 유저들의 평가가 많았다.
