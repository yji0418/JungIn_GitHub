CONSOLE GAME

주로 비디오 게임 등을 플레이하기 위한 전용 전자기기를 가리키는 말. 2000년대를 기점으로 플레이스테이션 2나 엑스박스 등의 게임기가 단순히 게임뿐만이 아니라 DVD/블루레이 시청 등의 멀티미디어 기기로도 사용되면서 그 범위가 넓어지고 있다.

  UNCHARTED 4

  개발사 : Naughty Dog

  유통사 : Sony Interactive Entertainment

  장르 : 액션 어드벤처

  심의등급 : 청소년 이용불가

  발매일(PS4 , PS4 Pro) : 2016년 5월 10일

  스토리 : 방대한 스토리 양으로 생략!

  평가 : PS의 독점출시작으로 GOTY 최다 수상작으로 선정되었다. 극강의 그래픽과 연출, 향상된 잠입 플레이, 시리즈를 훌륭하게 마무리 짓는 스토리와 깊어진 캐릭터성 등이 긍정적으로 평가되고 있다.






  Just Dance Series

  개발사 : Ubisoft Entertainment S.A.

  유통사 : X

  장르 : 댄스형 음악 게임

  심의등급 : X

  발매일(Wii) : 2009년

  플레이 방식 : 콘솔 리모콘을 갖고 화면에 나오는대로 춤을 따라서 추는 방식으로 진행된다.

  평가 : 운동용으로 구매한 사람이 있을정도로 격렬한 움직임이 필요한 게임 중 하나이다. 노래 별로 다른 독특한 스테이지와 화려한 댄서들이 특징이다.



  HALO 5

  개발사 : 343 Industries

  유통사 : Microsoft Studios

  장르 : FPS

  심의등급(국내 기준) : 15세 이용가

  발매일(Windows) : 2015년 10월 27일

  스토리 : 세계관 특성상 방대한 스토리로 생략

  평가 : 환상적인 멀티플레이와 부족한 싱글플레이. 역대 최악의 헤일로 라는 평가를 받은 헤일로4보다 낮은 점수를 기록했다. 하지만 전작에 비해 다양한 요소가 추가되고 무기들의 밸런스 조절이 잘 되어있다고 한다.
